NhoCSS

«Nho» significa “piccolo” in vietnamita — design minimal, pesi ridotti.

NhoCSS è un micro-framework CSS utility-first pensato per progetti leggeri, mashup e integrazioni dove serve solo l'essenziale: spacing, grid, flex e poche utilities. Questo sito è una documentazione WIP (work in progress).

About

WIP — questa documentazione è un mock. Quando il progetto sarà pronto, aggiornerò le API e gli esempi reali.

NhoCSS è un set di utility CSS minimale, pensato per essere incluso facilmente via file singolo (o CDN). Filosofia: piccolo, componibile, senza dipendenze.

Install

Disponibile come file singolo nella cartella dist/:

  • dist/nho.css — file leggibile per sviluppo
  • dist/nho.min.css — versione minimizzata per produzione

Esempio di inclusione:

<link rel="stylesheet" href="https://tuo-username.github.io/nho-css/dist/nho.min.css">

Reset

NhoCSS include un reset minimale per uniformare comportamento tra browser. Esempio base incluso in _reset.scss: 

/* reset minimo */
* { margin: 0; padding: 0; box-sizing: border-box; }
body { line-height: 1.5; font-family: 'Be Vietnam Pro', sans-serif; }

Se usi Bootstrap/Tailwind non è necessario includere questo reset (sono già presenti i rispettivi resets).

Variabili (design tokens)

NhoCSS espone poche variabili per mantenere il framework minimale. Definite in _variables.scss: 

$color-primary: #41b883;   /* verde Vue (accent) */
$color-black:   #000000;
$color-white:   #ffffff;
$color-gray:    #999999;

Usa queste variabili per personalizzare tema e colori, oppure aggiungi override nel tuo progetto.

Utilities

Set minimo di utility disponibili (spazio, testo, colore):

  • Spacing: .m-1, .m-2, .p-1, .p-2, etc.
  • Text: .text-left, .text-center, .text-right, .fw-bold
  • Color: .text-primary, .bg-primary, .bg-gray

Esempio rapido:

Questo paragrafo usa .m-2 .p-2 .text-center

Grid

NhoCSS fornisce una row/col grid in stile Bootstrap:

  • .row — container con 12 colonne di base (grid-template-columns: repeat(12,1fr))
  • .col — colonna automatica (si divide lo spazio disponibile)
  • .col-1 … .col-12 — colonne che spanano N unità

Esempio:

<div class="row">
  <div class="col-4">4/12</div>
  <div class="col-4">4/12</div>
  <div class="col-4">4/12</div>
</div>

<div class="row">
  <div class="col">auto</div>
  <div class="col">auto</div>
  <div class="col">auto</div>
</div>

Flex

Utility flex minimal ma componibile (ispirate a Bootstrap):

  • .flex — display:flex
  • .flex-row / .flex-column — direzione
  • .flex-wrap — wrap
  • .justify-center, .justify-between, ... — allineamento orizzontale
  • .items-center, .items-start — allineamento verticale
  • .flex-center — scorciatoia per centrare orizz. e vert.

Esempio:

<div class="flex justify-between items-center">
  <div>left</div>
  <div>right</div>
</div>

Tipografia

NhoCSS usa di default il font Be Vietnam Pro (import incluso in dist/nho.css). Utility disponibili:

  • .text-xs … .text-2xl — dimensioni
  • .fw-light, .fw-normal, .fw-semibold, .fw-bold
  • .leading-normal, .leading-tight — line-height

Responsive

Breakpoints minimi: sm (576px), md (768px), lg (992px). Il file _responsive.scss fornisce varianti base per colonne e spacing (es. .col-6@md, .m-2@sm).

Esempio (mobile-first):

@media (min-width: 768px) {
  .col-6\@md { grid-column: span 6; }
}
