About NhoCSS

NhoCSS is a minimal, mobile-first CSS utility framework designed for fast prototyping and clean interfaces. Inspired by the simplicity of utility-first CSS, NhoCSS provides only the essential classes for layout, spacing, grid, flexbox, typography, and colors. Lightweight and dependency-free, it is ideal for small projects, mashups, and integrations where you need just the basics.

Grid System

NhoCSS provides a minimal grid system using the .grid class for the container and .col for columns. The grid is mobile-first and uses CSS Grid with 12 equal columns by default. You can create responsive layouts easily with breakpoint modifiers like @lg.

Example

                
<div class="grid">
    <div class="col">Column 1</div>
    <div class="col">Column 2</div>
    <div class="col">Column 3</div>
</div>
  • .grid: grid container with 12 equal columns
  • .col: grid column, divides space equally by default
  • .col-span-{n}@{breakpoint}: span n columns at breakpoint

The grid is mobile-first, so styles apply to all screen sizes unless overridden by breakpoint-specific classes like @lg.

Flexbox Utilities

NhoCSS provides simple flexbox utilities to create flexible layouts. Use the .flex class to enable flexbox, and combine with direction classes like .flex-row, .flex-column, and responsive variants such as .flex-row@lg.

Example

                
<div class="flex flex-row gap-3">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>

<div class="flex flex-column@lg gap-2">
    <div>Item A</div>
    <div>Item B</div>
    <div>Item C</div>
</div>
  • .flex: enables flexbox container
  • .flex-row, .flex-column: flex direction
  • .flex-row@lg, .flex-column@lg: responsive flex direction at large breakpoint
  • .gap-{n}: gap between flex items

These utilities are mobile-first, so you can define default flex directions for small screens and adjust layouts for larger screens using breakpoint modifiers.

Typography

NhoCSS includes styling utilities to easily style text elements. Use classes for font size, weight, alignment, and more to create readable and attractive text.

Example

        
<p class="text-lg fw-semibold text-center">This is a large, semi-bold, centered paragraph.</p>
<h1 class="text-3xl fw-bold">Heading 1</h1>
  • .text-{size}: sets font size (e.g. .text-lg, .text-2xl)
  • .fw-{weight}: sets font weight (e.g. .fw-bold, .fw-semibold)
  • .text-{align}: text alignment (e.g. .text-center, .text-left)
  • .text-{color}: text color utility

Combine these classes to customize typography responsively and consistently across your project.

Colors

NhoCSS provides color utilities for text, backgrounds, and borders. Use predefined color classes to maintain consistent color schemes throughout your site.

Example

        
<div class="bg-primary text-white p-3">Primary background with white text</div>
<span class="text-secondary">Secondary colored text</span>
  • .text-{color}: sets text color (e.g. .text-primary, .text-secondary)
  • .bg-{color}: sets background color (e.g. .bg-primary, .bg-gray)
  • .border-{color}: sets border color

Use these classes to apply colors consistently and support theming with ease.

Margin & Padding

Quickly add spacing to elements with margin and padding utilities. Use shorthand classes to control spacing on all sides or specific directions.

Example

        
<div class="m-4 p-3">Margin 4 and padding 3</div>
<div class="mt-2 mb-2 px-4">Vertical margin 2 and horizontal padding 4</div>
  • .m-{n}: margin on all sides
  • .mt-{n}, .mb-{n}, .ml-{n}, .mr-{n}: margin top, bottom, left, right
  • .p-{n}: padding on all sides
  • .pt-{n}, .pb-{n}, .pl-{n}, .pr-{n}: padding top, bottom, left, right

Spacing values are typically numeric scale units that correspond to fixed pixel values for consistency.
